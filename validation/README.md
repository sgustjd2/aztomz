# 한끗 — 수요 검증 (경로 A) 실행 키트

**목표:** 코드 0줄. 지인 10명에게 "그 인스타 맛집/디저트, 광고야 진짜야?"를 손으로 분석해주고,
사람들이 **(a) 남에게 공유하나 / (b) "이것도 봐줘"라고 또 물어보나**를 관찰한다.
이게 "사람들이 신뢰 분석을 진짜 원하나"(= 이 서비스의 가장 위험한 가정)를 검증한다.

## 이번 주 순서

1. **모집** — 아래 메시지를 친구·지인 12~15명에게 보낸다(10건 채우려면 여유분 필요).
2. **분석** — 받은 항목마다 [`template.md`](template.md)를 복사해 채운다(1건 10분 목표). 예시는 [`example.md`](example.md).
3. **전달** — 완성된 1장을 보낸 사람에게 돌려준다.
4. **관찰** — 아래 추적 시트에 결과를 기록한다. *분석 품질보다 사람 반응이 핵심.*

## 합격선 (이번 주~2주)

- **(a) 최소 3명이 자발적으로 남에게 공유** OR **(b) 최소 3명이 다른 항목도 또 물어봄**
- 둘 다 안 나오면 → 쐐기/메시지 재검토 (C 빌드 보류)
- 이 10건은 나중에 hermes 자동 판정의 **정답셋**이 되니 [`template.md`](template.md) 양식으로 일관되게 기록.

## 모집 메시지 (카톡/DM 복붙용)

> 혹시 요즘 인스타나 틱톡에서 자주 본 **맛집이나 디저트 하나**만 알려줄래?
> (이름 + 어디서 봤는지: 인스타/유튜브/지도 등)
> 내가 그게 **진짜 갈 만한지 vs 광고·체험단 거품인지** 분석해서 한 장으로 정리해 보내줄게 🙂

변형(더 구체적으로 받고 싶을 때):
> "가보고 싶은데 광고인지 진짜인지 헷갈리는 곳" 있으면 딱 하나만 던져줘. 링크 있으면 더 좋고.
