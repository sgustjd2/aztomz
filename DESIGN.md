# 한끗 디자인 시스템 (v2 — 에디토리얼)

> 캐릿(careet.net)의 **에디토리얼 매거진 느낌만** 차용해 재창조. 색·폰트·로고·레이아웃 복제 금지.
> 구현: `assets/styles.css` (모든 페이지 공유). 데이터/헬퍼: `assets/app.js`.
> v1(코럴 카드 피드)에서 "AI 티" 지적 → v2에서 매거진으로 재설계.

## AI 슬롭을 없앤 핵심 결정
- ❌ **이모지 썸네일 제거** → ✅ 타이포그래픽 커버(색면 + 큰 한글 제목 + 회전 스탬프, 대각선 텍스처).
- ❌ 똑같은 카드 세로 반복 → ✅ **마스트헤드 → 피처드 → 랭킹 인덱스 → 트렌드 스트립**의 비대칭 매거진 리듬.
- ❌ 동글동글 큰 radius → ✅ 각진 인쇄 radius(7px), 얇은 룰, 강한 2px 섹션 룰.
- ✅ 세리프(Georgia) 액센트: 발행 정보·섹션 번호(01/02)·점수 큰 숫자·풀쿼트 → "인쇄/매거진" 질감.

## 컬러 토큰
| 토큰 | 값 | 용도 |
|---|---|---|
| `--paper` | `#f3efe6` | 뉴스프린트 배경 |
| `--ink` | `#17140f` | 본문/제목 |
| `--ink-2` | `#5c554c` | 보조 |
| `--line` | `#ddd5c6` | 룰/보더 · `--rule #17140f` 강조 룰 |
| `--accent` | `#e8431f` | 브랜드 + 광고 의심도(버밀리언, 캐릿 라임 회피) |
| `--green` | `#1c7a4a` | 실제 후기 신뢰도 · 긍정/추천 |
| `--amber` | `#c98a2b` | 중립 라벨 |

**의미 규칙:** 버밀리언=광고/주의, 초록=신뢰/만족. 깨지 말 것(초록을 브랜드색으로 쓰면 신뢰점수와 혼동 → 캐릿 라임 못 쓰는 이유).

## 타이포그래피
- `--sans`: Apple SD Gothic Neo → Pretendard → Noto Sans KR → Malgun Gothic (시스템 스택, **외부 CDN 0**).
- `--serif`: Georgia — 발행정보·섹션번호·점수 숫자·풀쿼트에만.
- 스케일: 홈 H1 42/900, 상세 제목 30/900, 점수 숫자 40(serif), 섹션 22/900, 본문 14~15.

## 컴포넌트 (styles.css)
- **`.mast`**: 워드마크 한끗 + 세리프 발행라인 + 인증 인지 네비(로그인 상태=아바타). `app.js H.renderMast()`가 주입.
- **`.cover`** (`.cat-*` 색면): 타이포 커버. 이모지 금지. `.tall`=상세/피처드용.
- **`.featured`**: 2단(커버 + 본문 + 점수 칩).
- **`.idx-row`**: 랭킹 인덱스(세리프 번호 + 미니 막대).
- **`.score`**: 큰 serif 숫자 + 막대(광고=버밀리언, 신뢰=초록).
- **`.label`** good/warn/mid, **`.sat-badge`** pos/neg/mix(만족도 = 신뢰도와 별개 축, PRD 9.7 보정).
- **`.pull`**: 세리프 풀쿼트. **`.empty`**: 따뜻한 빈 상태(마크 + 제목 + 1차 액션).
- 폼/버튼/리뷰/토스트/저장버튼 전부 공유.

## 모션
- `.reveal` fade-up은 순수 강화: 기본 보임, JS+IO일 때만, 1.5초 안전장치 + reduced-motion 존중. backdrop-filter 미사용(렌더 안정성).
