\# PRD: AZ와 MZ는 한 끗 차이



\## 0. 프로젝트 한 줄 소개



\*\*AZ와 MZ는 한 끗 차이\*\*는 요즘 유행하는 말, 밈, 음식, 디저트, 핫플, 브랜드, 앱, 숏폼 트렌드를 1분 안에 이해할 수 있게 번역해주는 트렌드 큐레이션 웹/앱 서비스입니다.



단순히 “요즘 뭐가 유행이다”를 알려주는 것이 아니라,



\- 이게 뭔지

\- 왜 뜬 건지

\- 누가 좋아하는지

\- 지금 따라 해도 되는지

\- 아재처럼 보이지 않게 어떻게 말하면 되는지

\- SNS에서 많이 보이는 맛집/디저트가 광고인지 실제 후기인지



까지 함께 알려주는 것을 목표로 합니다.

\## 1. 서비스명



\## 1.1 메인 후보



\# AZ와 MZ는 한 끗 차이



\## 1.2 이름 의미



AZ와 MZ는 글자 하나 차이지만, 실제로는 트렌드를 아느냐 모르느냐에 따라 대화의 거리감이 생깁니다.



이 서비스는 그 한 끗 차이를 줄여주는 서비스입니다.



\## 1.3 서비스명 장점



\- 기억하기 쉽다.

\- AZ, MZ라는 키워드가 직관적이다.

\- 세대 차이를 무겁지 않게 풀 수 있다.

\- 밈, 음식, 디저트, 핫플, 브랜드, 말투까지 확장 가능하다.

\- “한 끗 차이”라는 표현이 서비스의 핵심 가치와 잘 맞는다.



\## 1.4 줄임말 후보



| 줄임말 | 느낌 | 추천도 |

|---|---|---:|

| 한끗 | 가장 자연스럽고 브랜드화 가능 | 높음 |

| AZMZ | 직관적이지만 다소 딱딱함 | 중간 |

| 끗차이 | 캐주얼하지만 발음이 약간 어색함 | 중간 |

| 요즘끗 | 요즘 트렌드 느낌 있음 | 중간 |

| 아엠지 | AZ와 MZ를 섞은 느낌, 장난스러움 | 낮음 |

| 한끗차 | 앱 이름으로는 약간 애매함 | 낮음 |

\## 2. 서비스 컨셉



\## 2.1 핵심 컨셉



요즘 유행하는 것들을 몰라서 대화에서 소외되거나, 뒤늦게 따라 하다가 어색해지는 사람들을 위해 트렌드를 쉽게 번역해주는 서비스입니다.



\## 2.2 핵심 문장



“이게 뭔데?”

“왜 뜬 건데?”

“나도 해도 돼?”

“이거 광고 아니야?”

“어떻게 말해야 안 어색해?”



이 5가지 질문에 답해주는 서비스입니다.



\## 2.3 사용자에게 주는 가치



| 가치 | 설명 |

|---|---|

| 이해 | 요즘 말, 밈, 음식, 디저트, 핫플을 쉽게 이해 |

| 판단 | 지금 따라 해도 되는 유행인지 판단 |

| 신뢰 | 광고성 콘텐츠인지 실제 후기인지 구분 |

| 소통 | 세대 차이로 생기는 대화 거리감 감소 |

| 발견 | 요즘 뜨는 음식, 디저트, 장소를 빠르게 발견 |

| 저장 | 나중에 볼 트렌드, 맛집, 디저트를 저장 |

\## 3. 문제 정의



\## 3.1 사용자가 겪는 문제



요즘 트렌드는 대부분 인스타그램, 틱톡, 유튜브 쇼츠, X, 블로그, 커뮤니티, 뉴스레터 등에 흩어져 있습니다.



사용자는 다음과 같은 문제를 겪습니다.



1\. 유행어를 들어도 정확한 뜻을 모른다.

2\. 요즘 뜨는 음식이나 디저트를 뒤늦게 알게 된다.

3\. SNS에서 자주 보이는 맛집이 진짜 맛집인지 광고인지 구분하기 어렵다.

4\. 검색해도 너무 많은 정보가 나오고, 핵심만 정리된 곳이 없다.

5\. 트렌드를 따라 하고 싶지만 어색하거나 늦어 보일까 봐 걱정된다.

6\. 부모 세대, 직장 상사, 자영업자, 마케터는 MZ/Z세대 트렌드를 따라가기 어렵다.

7\. 맛집 추천 콘텐츠는 많지만 광고, 협찬, 체험단, 실제 후기가 섞여 있어 신뢰하기 어렵다.



\---



\## 4. 제품 목표



\## 4.1 핵심 목표



요즘 유행하는 말, 밈, 음식, 디저트, 핫플을 1분 안에 이해하고, 광고성 콘텐츠와 실제 후기를 구분할 수 있게 만든다.



\## 4.2 세부 목표



| 목표 | 설명 |

|---|---|

| 트렌드 번역 | 어려운 트렌드를 쉬운 말로 설명 |

| 유행 단계 판단 | 지금 뜨는 중인지, 피크인지, 끝물인지 표시 |

| 광고 의심 분석 | SNS 노출이 많은 맛집/디저트의 광고 가능성 분석 |

| 실제 후기 분석 | 리뷰 패턴, 방문 후기, 재방문 언급 등을 기반으로 신뢰도 판단 |

| 음식/디저트 큐레이션 | 요즘 뜨는 음식과 디저트를 한곳에서 모아보기 |

| 세대 간 소통 지원 | AZ 사용자가 MZ/Z세대 문화를 이해할 수 있게 도움 |

| 재방문 유도 | 매일 또는 매주 업데이트되는 트렌드 랭킹 제공 |

\## 5. 타깃 사용자



\## 5.1 1차 타깃: AZ 사용자



요즘 유행을 알고 싶지만 직접 인스타그램, 틱톡, 유튜브 쇼츠를 깊게 보지는 않는 사람입니다.



예시:



\- 30\~50대 직장인

\- 직장 상사

\- 학부모

\- 교사

\- 자영업자

\- 프랜차이즈 사장님

\- 마케터

\- 브랜드 담당자



주요 니즈:



\- 요즘 말의 뜻을 알고 싶다.

\- 젊은 직원, 자녀, 학생과 자연스럽게 대화하고 싶다.

\- 유행하는 음식과 디저트를 알고 싶다.

\- 이 트렌드를 지금 따라 해도 되는지 알고 싶다.

\- SNS 맛집이 진짜인지 광고인지 알고 싶다.



\## 5.2 2차 타깃: 20\~30대 트렌드 소비자



트렌드를 잘 알지만, 요즘 뜨는 음식, 디저트, 맛집, 핫플을 빠르게 모아보고 싶은 사람입니다.



주요 니즈:



\- 이번 주에 뭐 먹으러 갈지 알고 싶다.

\- 요즘 뜨는 디저트를 빠르게 보고 싶다.

\- 친구와 공유할 만한 장소를 찾고 싶다.

\- SNS에서 자주 보이는 곳이 실제로 갈 만한지 알고 싶다.



\## 5.3 3차 타깃: 마케터/자영업자/브랜드 담당자



요즘 트렌드를 콘텐츠, 메뉴 개발, 마케팅에 활용하고 싶은 사람입니다.



주요 니즈:



\- 요즘 어떤 키워드가 뜨는지 알고 싶다.

\- 음식/디저트 트렌드를 메뉴 개발에 참고하고 싶다.

\- 특정 맛집이나 브랜드가 왜 뜨는지 알고 싶다.

\- 광고성 바이럴과 실제 유행을 구분하고 싶다.

\# 8.1 MZ 트렌드 번역기



\## 기능 설명



사용자가 유행어, 밈, 음식 이름, 디저트 이름, 브랜드명, 앱 이름, 장소명을 검색하면 쉽게 해석해주는 기능입니다.



\## 검색 예시



\- 추구미

\- 럭키비키

\- 원영적 사고

\- 두바이 초콜릿

\- 탕후루

\- 요아정

\- 올드머니룩

\- 러닝크루

\- 도파민

\- 로우앵글 건배샷

\- 성수 팝업

\- 말차 디저트



\## 결과 예시



\### 검색어: 두바이 초콜릿



\- 한 줄 요약: 피스타치오 크림과 바삭한 카다이프 식감으로 유명해진 초콜릿 디저트

\- 왜 떴나: 숏폼에서 자르는 장면, 바삭한 소리, 고급스러운 비주얼이 퍼지며 유행

\- 누가 좋아하나: 디저트 좋아하는 10\~30대, 인증샷 소비자, 카페 투어층

\- 유행 단계: 피크 이후 안정화

\- 따라 해도 되나: 먹는 것은 자연스럽지만 “이게 요즘 대세라며?”라고 말하면 살짝 늦어 보일 수 있음

\- 자연스러운 표현: “그 피스타치오 들어간 바삭한 초콜릿 말하는 거지?”

\- 광고 의심도: 중간

\- 실제 후기 신뢰도: 브랜드/매장별 차이 큼

\## 기능 설명

\## 주요 필터



| 필터 | 예시 |

|---|---|

| 카테고리 | 음식, 디저트, 음료, 카페, 편의점, 술, 야식 |

| 유행 단계 | 뜨는 중, 피크, 안정화, 끝물 |

| 지역 | 서울, 경기, 부산, 대구, 대전, 광주, 제주 |

| 가격대 | 5천원 이하, 1만원대, 2만원대, 고가 |

| 목적 | 데이트, 친구, 혼자, 가족, 회식, 인증샷 |

| 난이도 | 쉽게 먹을 수 있음, 웨이팅 필요, 예약 필수 |

| 신뢰도 | 광고 의심 낮음, 중간, 높음 |

| 플랫폼 | 인스타, 틱톡, 유튜브, 블로그, 네이버 검색 |



\## 음식/디저트 카드 예시



\### 말차 디저트



\- 분류: 디저트

\- 유행 단계: 상승

\- 특징: 진한 녹차맛, 고급스러운 색감, 카페 신메뉴로 확산

\- 먹어볼 만한 사람: 단맛보다 쌉싸름한 맛을 좋아하는 사람

\- 주의: 매장마다 맛 차이가 크고, 일부는 비주얼 중심일 수 있음

\- 광고 의심도: 중간

\- 실제 후기 신뢰도: 리뷰 내 맛 언급이 많으면 높음

\## 9. 광고 의심도 및 실제 후기 분석 기능



\## 9.1 기능 목적



SNS에서 많이 보인다고 해서 반드시 좋은 맛집이나 디저트는 아닙니다.



오히려 SNS에 과도하게 많이 노출되는 경우, 광고, 협찬, 체험단, 바이럴 마케팅일 가능성이 있습니다.



따라서 한끗은 단순 인기 순위가 아니라,



\- SNS 화제성

\- 광고 의심도

\- 실제 후기 신뢰도

\- 방문 만족도



를 분리해서 보여줍니다.



\---



\## 9.2 핵심 판단 원칙



\## 중요한 관점



SNS 노출이 많을수록 무조건 좋은 것이 아니라, 광고일 확률도 함께 증가할 수 있습니다.



따라서 다음과 같이 해석합니다.



| 상태 | 해석 |

|---|---|

| SNS 노출 많음 + 실제 후기 좋음 | 진짜 인기 맛집 가능성 높음 |

| SNS 노출 많음 + 실제 후기 부족 | 광고/바이럴 가능성 있음 |

| SNS 노출 많음 + 비슷한 문구 반복 | 체험단/협찬 의심 |

| SNS 노출 적음 + 실제 후기 좋음 | 숨은 맛집 가능성 |

| SNS 노출 적음 + 리뷰도 적음 | 판단 보류 |

| SNS 노출 많음 + 평점 변동 큼 | 기대와 실제 경험 차이 가능성 |

\## 9.3 광고 의심도 지표



광고 의심도는 아래 요소를 기반으로 산정합니다.



| 지표 | 설명 | 광고 의심 증가 조건 |

|---|---|---|

| 동일 문구 반복 | 여러 게시물에서 비슷한 표현 반복 | “인생맛집”, “무조건 가야 함” 등 반복 |

| 동일 해시태그 반복 | 협찬성 해시태그 패턴 | #협찬, #제공, #광고, #체험단 등 |

| 짧은 기간 집중 노출 | 특정 기간에 포스팅이 몰림 | 오픈 직후 1\~2주 내 과도한 노출 |

| 인플루언서 비중 | 일반 사용자보다 인플루언서 게시물이 많음 | 팔로워 많은 계정 중심 확산 |

| 리뷰 내용의 구체성 부족 | 맛, 가격, 웨이팅 등 실제 경험 설명 부족 | “분위기 미쳤다”, “꼭 가야 함”만 반복 |

| 사진 구도 유사성 | 같은 메뉴, 같은 포토존, 같은 구도 반복 | 마케팅 가이드 가능성 |

| 지도 리뷰와 SNS 괴리 | SNS는 핫하지만 지도 리뷰가 적거나 낮음 | 실제 방문 경험 부족 가능성 |

| 부정 후기 삭제 의심 | 부정적 리뷰가 거의 없고 문구가 과하게 긍정적 | 과도한 관리 가능성 |

\## 9.4 실제 후기 신뢰도 지표



실제 후기 신뢰도는 아래 요소를 기반으로 산정합니다.



| 지표 | 설명 | 신뢰도 증가 조건 |

|---|---|---|

| 구체적인 맛 표현 | 달다, 짜다, 느끼하다, 식감 등 실제 경험 묘사 | 구체적일수록 신뢰 증가 |

| 가격 만족도 언급 | 가격 대비 만족/불만족 여부 | 가격 평가가 다양하게 존재 |

| 웨이팅 경험 언급 | 실제 방문자가 겪은 대기 시간 | 시간대별 정보가 있으면 신뢰 증가 |

| 재방문 의사 | “또 갈 듯”, “재방문”, “다음엔 다른 메뉴” | 재방문 언급 많으면 긍정 |

| 단점 언급 | 장점뿐 아니라 단점도 함께 언급 | 균형 잡힌 리뷰는 신뢰 증가 |

| 일반 사용자 리뷰 비중 | 팔로워 적은 일반 계정/지도 리뷰 | 일반 사용자 비중 높으면 신뢰 증가 |

| 리뷰 기간 분산 | 특정 기간이 아니라 꾸준히 리뷰 발생 | 장기적으로 안정적이면 신뢰 증가 |

| 지도 리뷰와 SNS 반응 일치 | SNS와 지도 리뷰가 비슷한 평가 | 일치할수록 신뢰 증가 |

\## 18. 사용자 시나리오



\## 18.1 시나리오 1: AZ 사용자가 유행어를 검색



1\. 사용자가 “럭키비키”를 검색한다.

2\. 한 줄 요약을 확인한다.

3\. 왜 유행했는지 본다.

4\. 실제 사용 예시를 확인한다.

5\. 어색한 표현과 자연스러운 표현을 비교한다.

6\. 자녀나 친구에게 공유한다.



\---



\## 18.2 시나리오 2: 사용자가 SNS 맛집의 광고 여부를 확인



1\. 사용자가 인스타에서 자주 보이는 카페 이름을 검색한다.

2\. 한끗에서 해당 장소의 SNS 화제성을 확인한다.

3\. 광고 의심도와 실제 후기 신뢰도를 본다.

4\. 최종 판단 라벨을 확인한다.

5\. 방문 목적에 맞는 추천 여부를 확인한다.

6\. 지도 앱으로 이동하거나 저장한다.



\---



\## 18.3 시나리오 3: 20대 사용자가 이번 주 디저트를 확인



1\. 홈에서 “이번 주 뜨는 디저트”를 클릭한다.

2\. 말차 디저트, 피스타치오 디저트, 대왕 쿠키 등을 확인한다.

3\. 지역 필터를 적용한다.

4\. 광고 의심도가 낮고 실제 후기 신뢰도가 높은 곳을 고른다.

5\. 친구에게 공유한다.

6\. 주말에 방문한다.



\---



\## 18.4 시나리오 4: 마케터가 트렌드 소재를 찾음



1\. “요즘 숏폼 트렌드” 카테고리에 접속한다.

2\. 급상승 키워드를 확인한다.

3\. 특정 트렌드의 유행 배경을 읽는다.

4\. 브랜드가 활용해도 되는지 확인한다.

5\. 광고성으로 보이지 않는 활용법을 참고한다.

\## 19. 기능 요구사항



\## 19.1 검색



\- 사용자는 키워드를 입력할 수 있다.

\## 19.4 음식/디저트 목록



\- 음식/디저트는 이미지 중심 카드로 보여준다.

\- 유행 단계, 가격대, 먹어볼 만한 사람, 주의점을 보여준다.

\- 관련 음식점이나 브랜드가 있으면 연결한다.

\- 광고 의심도와 실제 후기 신뢰도를 표시한다.



\---



\## 19.5 음식점/핫플 목록



\- 음식점은 지역, 메뉴, 가격, 웨이팅, 추천 목적 기준으로 필터링 가능해야 한다.

\- 광고 의심도와 실제 후기 신뢰도 기준으로 정렬할 수 있어야 한다.

\- 네이버지도/카카오맵/구글맵 연결을 제공한다.



\---



\## 19.6 공유



\- 모든 상세 페이지는 공유 가능해야 한다.

\- 카카오톡, 링크 복사, 이미지 저장을 지원한다.

\- 공유용 제목은 자동 생성한다.



예시:



“요즘 두바이 초콜릿이 왜 떴는지 1분 요약”

“이 카페 광고일까 진짜일까?”

“AZ와 MZ는 한 끗 차이: 럭키비키 편”

두바이 초콜릿은 피스타치오 크림과 바삭한 식감 때문에 숏폼에서 확 뜬 디저트예요. 자르는 장면이 예쁘고 소리도 좋아서 릴스에서 많이 퍼졌습니다.

좋은 예:



SNS에서 비슷한 구도와 유사한 칭찬 문구가 반복되어 광고/체험단 가능성이 있습니다. 다만 지도 리뷰에서는 맛과 재방문 의사도 일부 확인되어, 분위기 목적 방문이라면 고려해볼 만합니다.

| MZ어 사전 | 단어 뜻만 있음 | 사용 상황, 말투, 음식, 핫플까지 연결 |
